Introdução sobre Angular

O Angular é um framework JavaScript amplamente utilizado para construir aplicações web dinâmicas e SPA (Single Page Applications).

Versões

A versão atual do Angular é X.Y.Z. Mantenha-se atualizado com as últimas versões para aproveitar as melhorias e correções de bugs.

Instalando o Angular CLI

O Angular CLI (Command Line Interface) é uma ferramenta que facilita a criação e gerenciamento de projetos Angular.
$ npm install -g @angular/cli

Executando aplicações em Angular

Navegue para o diretório do projeto e inicie a aplicação:
$ cd nome-do-projeto
$ ng serve

Acesse a aplicação no seu navegador em http://localhost:4200.
